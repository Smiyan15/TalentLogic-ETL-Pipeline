TalentLogic: End-to-End Recruitment ETL Pipeline
Tech Stack: Python (Pandas), Snowflake (Cloud Data Warehouse), Tableau Public.

Project Goal: Automate the scoring and categorization of job candidates to reduce manual HR audit time.

Extraction: Processed raw candidate data using Python.

Transformation: Applied business logic to rank candidates as "Top Tier" or "Qualified."

Loading: Orchestrated data ingestion into a Snowflake staging table.

Visualization: Built an executive dashboard in Tableau for real-time talent auditing.
